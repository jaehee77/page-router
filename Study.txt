# \_app.tsx (app 컴포넌트의 역할)

## export default function App({ Component, pageProps }: AppProps)

```
Component : pages 폴더내에 있는 컴포넌트를 불러옴
즉, 페이지 역할을 할 컴포넌트를 받아옴
pageProps: 페이지 컴포넌트에 정의되어 있는 props를 받아옴
```
